## 개요
- 부동산 거래 시, 개인이 필요한 정보를 얻는 과정의 불편함을 개선
- 부동산 거래 관련 공공데이터를 지도 기반으로 제공해, 쉽게 정보에 접근할 수 있도록 함
- 가격비교, 시세확인, 질의응답, 내역조회 등의 기능 제공
<br></br>
